# Game Sounds

Ce dossier contient les effets sonores du jeu.

## Fichiers Requis

- `game-start.mp3` - Son de début de partie (1-2s)
- `move.mp3` - Son de mouvement de pion (0.2s)
- `capture.mp3` - Son de capture de pion (0.3s)
- `victory.mp3` - Son de victoire (2-3s)
- `defeat.mp3` - Son de défaite (2s)

## Sources Recommandées (Libres de Droits)

1. **Freesound.org** - https://freesound.org/
2. **Mixkit.co** - https://mixkit.co/free-sound-effects/
3. **Zapsplat.com** - https://www.zapsplat.com/
4. **Pixabay** - https://pixabay.com/sound-effects/

- **defeat**: "lose", "fail", "game over", "sad"

## Format

- Format: MP3
- Qualité: 128-192 kbps
- Durée: Courte (< 3s)
- Volume: Normalisé

## Fichiers Temporaires

Pour le développement, vous pouvez utiliser des sons de test ou générer des bips simples.
Les sons seront chargés automatiquement par le hook `useGameSounds`.
